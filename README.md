# docseeker

## Project Structure

```
├── app
|   ├── controller      # API Controllers (handling request and response)
|   ├── model           # Model for database
|   ├── routes          # API resources route
|   ├── view            # For User interface
|       ├── templates   # Section of the page that will be used on more than one page (ex: navbar)
├── db                  # Database initializer
├── lib                 # Helper
├── res                 # Css, image and Javascript container
|   ├── css
|   ├── js
|   ├── image
```
